## Getting Started

To get a local copy up and running follow these simple example steps.

> - Run `git clone https://github.com/Nuri1977/recipe-application`
> - Run `cd recipe-application` to change to project root directory
> - Run `bundle install`
> - Run `rails db:create`
> - Run `rails db:migrate`
> - Run `npm install`
> - Run `rails s` to run the app
> - Opn `http://localhost:3000/` on your browser
